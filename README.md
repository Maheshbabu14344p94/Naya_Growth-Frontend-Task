# Sere Innovations Landing Page

Single-page responsive frontend implementation for the NayaGrowth Frontend Intern task.

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## How to Run Locally
1. Open the project folder.
2. Open `index.html` in a browser.

## Implementation Note
### Overall approach
I focused on turning the business brief into a clear communication flow for practical users: problem -> solution -> feature clarity -> trust -> action.

### Main design decisions
- Used a grounded green + neutral visual system to align with agri-tech trust.
- Kept language direct and non-technical for low-to-moderate technical familiarity.
- Built section hierarchy to mirror farm-level decision questions: What is this? Why does it matter? Can I trust it? What do I do next?
- Added a strong final inquiry CTA with a simple lead form UI.

### Assumptions made
- INR 10,000 is presented as approximate positioning, not a fixed universal price.
- Contact details are placeholders and should be replaced with real business information.

### What I would improve with more time
- Add bilingual content support (English + regional language options).
- Add lightweight analytics events for CTA and inquiry form interactions.
- Introduce optimized illustration assets with performance budgets.
